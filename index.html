CSS stands for Cascading Style Sheets.

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.

HTML stands for Hypertext Markup language.

  • HTML is the most basic building block of Web
  • HTML mostly uses Tags and short codes
  • HTML is the language used to create,structure,design websites

JS stands for JavaScript

  • Js used to make games and apps
  • Js is a scripting language
  • Js is an advanced programming language used to make websites more interactive and dynamic

Python is a high level , general-purpose programming language

PHP stands for PHP:Hypertext Preprocessor
